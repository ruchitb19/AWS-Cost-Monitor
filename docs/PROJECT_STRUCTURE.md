### `scripts/` Folder

#### `cost_analyzer.py` (Core Logic)
**What it does:**
1. Fetches last 30 days of AWS costs from Cost Explorer API
2. Calculates daily totals and per-service breakdown
3. Detects anomalies using 7-day rolling average
4. Stores everything in DynamoDB with TTL
5. Exports formatted data as JSON for dashboard

**Key functions:**
```python
fetch_aws_costs(days=30)          # Get data from AWS
detect_anomaly(cost, date)         # Flag unusual spending
analyze_and_store(cost_data)       # Process & save to DB
export_to_json(output_file)        # Generate dashboard data
```

**Run it:**
```bash
python scripts/cost_analyzer.py
```

**Output:**
- Prints to console (progress updates)
- Creates `dashboard/data.json` (input for dashboard)
- Stores in DynamoDB (permanent record)

#### `local_data_generator.py` (Testing)
**What it does:**
- Generates realistic mock AWS cost data
- No AWS credentials needed
- Perfect for testing without touching real AWS

**Run it:**
```bash
python scripts/local_data_generator.py
```

**Generates:**
- `mock_cost_data.json` (raw mock data)
- `dashboard/data.json` (ready for dashboard)
